# BoostCart Final Project

## Overview

BoostCart is a web application that lets users search Amazon products and view detailed information about them using a REST API.

The project was built using HTML, Bootstrap, CSS, JavaScript, and jQuery.

## Pages

### index.html
Shows featured ecommerce products when the page loads.

### search.html
Allows the user to search for products and filter results.

### details.html
Displays detailed information about a selected product.

## Features

- Product search
- Product filtering
- Product sorting
- Product details page
- Dynamic API data
- Error handling

## How To Use

1. Open the homepage.
2. Go to the search page.
3. Search for a product.
4. Click View Details to see more information.
